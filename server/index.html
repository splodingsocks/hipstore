Here's the very simple API documentation for the very simple Hipstore API (this is a demo, after all.)

Types

There's only one, and that's the product:

Product

Here it is in typescript: 
    
      class Product {
        name: string
        price: number
        image: string
        id: string
      }
And here's a JSON sample: 
    
      {
        "id": "1",
        "name": "Jame Octavio",
        "price": 850,
        "image": "aaron-alvarado-25213.jpg"
      }

Endpoints

GET /api/products

Returns: A list of products

GET /api/cart

Returns: A list of products

POST /api/cart/:id

Takes a string ID at the end of the path as the only parameter. Adds that product to the cart if it exists. Returns a list of products

DELETE /api/cart/:id

Takes a string ID at the end of the path as the only parameter. Removes that product from the cart if it is in the cart. Returns a list of products
